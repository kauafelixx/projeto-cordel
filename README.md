# Experiência Cordel Nordestina

![Preview visual do projeto](foto-folhas.jpg)

Uma experiência visual em HTML e CSS inspirada na literatura de cordel, na estética de folheto popular e na poesia nordestina brasileira. O projeto parte do exercício original do Curso em Vídeo e evolui para uma peça mais apresentável, com direção visual, responsividade e curadoria literária.

## Destaques

- Layout estático feito apenas com HTML e CSS.
- Acentuação corrigida no poema `Cordel Moderno`.
- Visual com textura de papel, contraste de tinta, tons quentes e referências de xilogravura.
- Leitura mais confortável em celulares, tablets e desktops.
- Seção de curadoria com Patativa do Assaré e Ariano Suassuna.
- Créditos e links para as fontes literárias usadas como referência.

## Tecnologias

- HTML5
- CSS3
- Google Fonts

## Como visualizar

Abra o arquivo `index.html` diretamente no navegador.

```bash
start index.html
```

# Northeastern Cordel Experience

A visual HTML and CSS experience inspired by Brazilian cordel literature, popular booklet aesthetics, and Northeastern poetry. The project starts from the original Curso em Vídeo exercise and turns it into a more polished piece, with stronger visual direction, responsiveness, and literary curation.

## Highlights

- Static layout built only with HTML and CSS.
- Fixed Portuguese encoding in `Cordel Moderno`.
- Paper texture, ink contrast, warm colors, and woodcut-inspired details.
- Improved reading experience across mobile, tablet, and desktop.
- Curated section featuring Patativa do Assaré and Ariano Suassuna.
- Credits and source links for literary references.

## Tech Stack

- HTML5
- CSS3
- Google Fonts

## How to View

Open `index.html` directly in your browser.

```bash
start index.html
```
